# Neon, Railway & Next.js

This is a [Next.js](https://nextjs.org/) app that uses [Neon Launchpad](https://neon.new/) to instantly generate a claimable database for you at [Neon](https://neon.com).

[![Deploy on Railway](https://railway.com/button.svg)](https://railway.com/deploy/)

## Features

- Everything you get with Next.js
- Full power of Postgres enhanced by Neon

## Local Development

Install your dependencies and run the dev command with your preferred package manager.

| Package Manager | Install        | Dev           |
| --------------- | -------------- | ------------- |
| `npm`           | `npm install`  | `npm run dev` |
| `pnpm`          | `pnpm install` | `pnpm dev`    |
| `yarn`          | `yarn install` | `yarn dev`    |
| `bun`           | `bun install`  | `bun dev`     |

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
